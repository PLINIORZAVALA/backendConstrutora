6. Crear una imagen adicional en un catálogo
Método HTTP :POST
URL :http://localhost:3000/catalogos/{id}/imagen
Cuerpo :
json

Copiar código
{
  "ruta_imagen": "nueva_imagen.jpg",
  "descripcion_imagen": "Imagen de un nuevo producto"
}
Ejemplo :http://localhost:3000/catalogos/10/imagen
7. Obtener todas las imágenes adicionales de un catálogo
Método HTTP :GET
URL :http://localhost:3000/catalogos/{id}/imagenes
Ejemplo :http://localhost:3000/catalogos/10/imagenes
8. Obtener una imagen adicional por su DNI
Método HTTP :GET
URL :http://localhost:3000/catalogos/imagen/{imagenId}
Ejemplo :http://localhost:3000/catalogos/imagen/13
9. Actualizar una imagen adicional
Método HTTP :PUT
URL :http://localhost:3000/catalogos/imagen/{imagenId}
Cuerpo :
json

Copiar código
{
  "ruta_imagen": "imagen_actualizada.jpg",
  "descripcion_imagen": "Imagen de producto actualizado"
}
Ejemplo :http://localhost:3000/catalogos/imagen/13
10. Eliminar una imagen adicional
Método HTTP :DELETE
URL :http://localhost:3000/catalogos/imagen/{imagenId}
Ejemplo :http://localhost:3000/catalogos/imagen/13
